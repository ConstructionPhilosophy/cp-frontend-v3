api endpoint – POST /users/{uid}/projects
Content-Type: multipart/form-data
form fields

title (string, required)

description (string, required)

location (string, optional)

tags[] (string, repeatable, optional)

images (file, repeatable, required — multiple files allowed)

sample form-data (conceptual)

title: Sustainable Pavement Trial
description: Full-depth reclamation demo with foam bitumen.
location: Pune
tags[]: pavement
tags[]: sustainability
images: <file1.jpg>
images: <file2.jpg>
